    Buttons

    Buttons have two parent classes in the SCSS file .btn and .button

    All of the colouration classes are given the same option, so you can mix-and-match the button classes however you prefer.

    .button-small also has a short-hand option of .btn-sm.

    HTML

    <button type="button" class="button">Button</button>
<button type="button" class="button button-sm">Button</button>
<button type="button" class="button button-small">Button</button>

<button type="button" class="btn">Button</button>
<button type="button" class="btn btn-sm">Button</button>
<button type="button" class="btn btn-small">Button</button>

    .button-primary buttons have a .button-blue shorthand class; they also have an even shorter .btn-blue option.

    Flags

    Flag Sprites

    Flag Sprites was made using the Stripe Flag Set release on Dribbble. It was then converted into a spritesheet setup (available on CodePen) and included into the framework.

    Flag Sprites can be used two ways, either as dedicated inlined images or as an adative to a pre-existing element.

    HTML

    <i class="flag flag-{COUNTRY_CODE}"></i>

    Patterns

    Patterns an API-based web app that leverages the power of GeoPatterning.

    This is best used as a placeholder or as means to add colour, texture and personality to an otherwise mundane page.

    Patterns change in dimensions, color and pattern based on the input.

    It's also available open-source!

    HTML (<img>)

    <img src="https://patterns.forcir.com/v1/patterns" alt="Pattern Block" />

    HTML

    <div class="pattern-block"></div>

    SCSS

    .pattern-block {
    @include pattern(patterns);
    height: 150px;
    width: 150px;
}
